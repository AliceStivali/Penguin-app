## Installazione



- Prima procedere all'installazione avete bisogno di un token per Amadeus API, registratevi ed inseritelo nel file: root.js



Clona il progetto

```bash
  git clone https://github.com/AliceStivali/Penguin-app.git
```

Installa le dependencies

```bash
  npm install 
```

Avvio

```bash
  npm start
```
